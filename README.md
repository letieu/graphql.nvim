# Graphql.nvim

Interactive Graphql client for neovim


> [!CAUTION]
> Currently, this plugin is under development. It may not work as expected.

## Installation

* With **lazy.nvim**
```lua
{
  "letieu/graphql.nvim",
}
```

## Usage

**Open**
```lua
require('graphql').open()
```
